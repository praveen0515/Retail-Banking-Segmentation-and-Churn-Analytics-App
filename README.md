# Customer Segmentation & Churn Pattern Analytics in European Banking

An interactive data science dashboard built with **Streamlit** and **Plotly** to visualize, segment, and track customer churn dynamics across European retail banking portfolios.

## 💻 Live Dashboard
[🔗 Click here to view the live interactive application](https://retail-banking-segmentation-and-churn-analytics-app.streamlit.app)

---

## 🎯 Project Overview & Objectives
Customer churn represents one of the largest hidden costs in retail banking, causing reduced lifetime value and revenue instability. This project provides systematic, segmentation-driven analytics to answer key business issues:
* **Overall Churn Metrics:** Quantify systemic churn exposure across the asset portfolio.
* **Geographic Risk Indexes:** Compare and pinpoint high-risk European regional markets.
* **Demographic Cross-Analysis:** Evaluate how customer age profiles, tenures, and gender distributions interact with retention.
* **Financial Profile Explorations:** Map account balances, product ownership frequencies, and credit worthiness against churn probabilities.

## 📦 Features & Functionality
* **Dynamic Sidebar Filters:** Slice the entire dataset by Country, Gender, and Member Activity status on the fly.
* **High-Value Customer Analysis:** Visualizations tracking revenue risk by exploring balance vs. credit score maps.
* **Product Ownership Insights:** Clear analysis highlighting shifts in churn probability based on total product holdings.
* **Granular Inspection:** A sliceable data explorer view to inspect filtered raw records directly.

## 🛠️ Tech Stack & Architecture
* **Language:** Python
* **Framework:** Streamlit (Live UI layout and state management)
* **Visualizations:** Plotly Express & Graph Objects (Interactive charts with live data recalculations)
* **Data Wrangling:** Pandas
